Working with Arrays in Java

Array Declaration

An array in Java is a data structure that allows you to store multiple values of the same type. Here's how you can declare a 1D and a 2D array in Java: 

            
                
// 1D Array Declaration

                int[] oneDimensionalArray = new int[5];

                
// 2D Array Declaration

                int[][] twoDimensionalArray = new int[3][4];

Taking Input Using Scanner Class

To take input in Java, you can use the Scanner class. Below is an example of how to take input for 1D and 2D arrays using the Scanner class: 

            
                import java.util.Scanner;

                Scanner scanner = new Scanner(System.in);

                
// 1D Array Input

                System.out.print("Enter the size of the array: ");
                int size = scanner.nextInt();
                int[] array = new int[size];

                System.out.println("Enter the elements of the array:");
                for (int i = 0; i < size; i++) {
                    array[i] = scanner.nextInt();
                }

               
 // 2D Array Input

                System.out.print("Enter the number of rows: ");
                int rows = scanner.nextInt();
                System.out.print("Enter the number of columns: ");
                int columns = scanner.nextInt();
                int[][] matrix = new int[rows][columns];

                System.out.println("Enter the elements of the 2D array:");
                for (int i = 0; i < rows; i++) {
                    for (int j = 0; j < columns; j++) {
                        matrix[i][j] = scanner.nextInt();
                    }
                }

                // Close the scanner
                scanner.close();

Taking Input Using command line argument

To take input in Java, you can use the command line argument . Below is an example of how to take input for 1D and 2D arrays using the Scanner class: 

            
                
//command line argument 1D

                public class CommandLineArray1D {
                    public static void main(String[] args) {
                        if (args.length == 0) {
                            System.out.println("Please provide array elements as command line arguments.");
                            return;
                        }
                
                        // Assuming all arguments are integers
                        int[] array = new int[args.length];
                
                        // Parse and store elements in the array
                        for (int i = 0; i < args.length; i++) {
                            array[i] = Integer.parseInt(args[i]);
                        }
                
                        // Display the elements of the array
                        System.out.print("1D Array elements: ");
                        for (int num : array) {
                            System.out.print(num + " ");
                        }
                    }
                }

                
                
//command line argument 2D

                public class CommandLineArray2D {
                    public static void main(String[] args) {
                        if (args.length == 0) {
                            System.out.println("Please provide 2D array elements as command line arguments.");
                            return;
                        }
                
                        // Assuming all arguments are integers
                        int rows = Integer.parseInt(args[0]);
                        int columns = Integer.parseInt(args[1]);
                
                        // Check if the number of arguments matches the size of the 2D array
                        if (args.length != 2 + rows * columns) {
                            System.out.println("Invalid number of arguments for the 2D array.");
                            return;
                        }
                
                        int[][] matrix = new int[rows][columns];
                
                        // Parse and store elements in the 2D array
                        for (int i = 0; i < rows; i++) {
                            for (int j = 0; j < columns; j++) {
                                matrix[i][j] = Integer.parseInt(args[2 + i * columns + j]);
                            }
                        }
                
                        // Display the elements of the 2D array
                        System.out.println("2D Array elements:");
                
                        for (int i = 0; i < rows; i++) {
                            for (int j = 0; j < columns; j++) {
                                System.out.print(matrix[i][j] + " ");
                            }
                            System.out.println();
                        }
                    }
                }
